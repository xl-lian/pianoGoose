# 🎹 pianoGoose · 用电脑键盘学钢琴

一个跑在 macOS 上、用电脑键盘练习弹钢琴的网页 App。纯前端、无需安装依赖、离线可用。

## 快速开始

**最简单：** 双击 `start.command`（首次可能需要在「系统设置 › 隐私与安全性」里允许运行），浏览器会自动打开。

**或者手动起一个本地服务器**（推荐，音频与 MIDI 更稳定）：

```bash
cd pianoGoose
python3 -m http.server 8000
# 然后浏览器打开 http://localhost:8000
```

> 也可以直接双击 `index.html` 用 file:// 打开，基本功能都能用。

## 键位映射（已预定义）

仿 GarageBand「音乐键入」布局，最贴近真实钢琴手感：

- **白键**（主键盘一排）：`A S D F G H J K L ; '` → C D E F G A B C D E F
- **黑键**（上方一排，错落对应）：`W E _ T Y U _ O P _ ]` → C# D# / F# G# A# / C# D# / F#
- **移八度**：`Z` 降八度，`X` 升八度（一排覆盖约 1.5 个八度，配合移八度可达全键盘）
- **空格**：在曲目 / 乐理模式下播放 / 暂停

屏幕底部「键位映射」一栏会实时显示每个电脑键当前对应哪个音。

## 三种模式

1. **自由演奏** — 随便弹，屏幕大字显示音名与唱名。
2. **弹奏曲目** — 落键瀑布（Synthesia 风格）。音符自上而下落，落到判定线时按对应键。
   - **等待模式**：音符落到线上会停下等你按对，最适合初学。
   - **试听**：先听一遍自动演奏。
   - 速度可调 0.5×～1.25×。
3. **乐理练习** — 音阶 / 和弦 / 和弦进行，自动生成带指法标注的练习谱，同样用瀑布呈现。

落下来的方块显示**真实音名 / 唱名 + 该按的电脑键 + 指法**——音名是主角，电脑键作为「按哪个键」的提示贴在落点上。可在底部「标注」里切换：键+音名 / 仅音名 / 唱名 / **五线谱** / 仅按键。

### 识谱练习（五线谱标注）

把「标注」切到 **五线谱**：落下的方块和每个琴键上都会画出**高音谱表上的音符位置**（含上下加线、升号），方块上同时保留电脑键提示。这样能把「五线谱位置 ↔ 键盘键 ↔ 钢琴键」对应起来练习识谱。

## 主题配色

顶栏「主题」可切换 **午夜 / 青碧 / 暮色 / 日光**（含浅色主题）四套配色，选择会自动记住。

### 用简谱添加自己的歌

在「弹奏曲目」面板底部展开 **＋ 用简谱添加自己的歌**，粘贴简谱即可练习任意曲子，并保存到曲库（存于浏览器本地）。简谱语法：

```
1-7 = do re mi fa sol la si    0 = 休止
'  高八度（可叠加 ''）          ,  低八度
#/b 升/降（写在数字前，如 #4）   .  附点(×1.5)
/  八分音符（每个 / 再 ×0.5）   -  把前一个音延长一拍
|  小节线（忽略）

例：1 1 5 5 | 6 6 5 - | 4 4 3 3 | 2 2 1 -   （小星星，调=C）
```

调（1= 哪个音）、速度都可在导入框里设置。

## 真实 MIDI 键盘

插上 USB-MIDI 键盘后会自动识别（需用 `http://localhost` 方式打开，浏览器才允许 Web MIDI）。顶栏右侧会显示设备名。

## 内置曲库

**公有领域旋律**（可放心练习）：小星星、欢乐颂、玛丽的小羊、两只老虎、生日快乐、致爱丽丝（开头）、茉莉花、铃儿响叮当、平安夜、伦敦桥、绿袖子、奇异恩典、友谊地久天长、卡农、王老先生有块地、康定情歌。

**华语流行 / 现代歌曲**：请用上面的「简谱导入」自行添加——粘贴该曲简谱即可精确练习并保存。（不内置是因为现代流行曲多为版权作品，且无法凭记忆保证旋律准确。）

## 技术栈

纯 HTML / CSS / 原生 JavaScript，Web Audio 合成音色（无采样文件），Web MIDI 可选。无构建步骤。

```
index.html
css/styles.css
js/
  core.js       音乐基础工具（音名/频率/MIDI）
  jianpu.js     简谱解析器（导入自定义曲）
  keymap.js     电脑键盘 → 音符 映射
  audio.js      Web Audio 合成钢琴
  theory.js     音阶/和弦/指法/进行
  songs.js      内置曲库
  piano.js      屏幕钢琴渲染
  waterfall.js  落键瀑布引擎
  midi.js       Web MIDI（可选）
  app.js        主控制器
```
